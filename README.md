# 🖥️ Parc Informatique - Gestion complète du parc matériel

----

**Objectifs :**  
- 🖱️ Suivi du matériel (PC, imprimantes…)  
- 👩‍💼 Gestion des employés  
- 📦 Affectation et retour du matériel  
- 📊 Filtrage et statistiques pour faciliter la prise de décision

## Les entités principales sont :

- **Materiel** : id, ref, type, marque, état, dateAchat  
- **Employe** : id, nom, service, email  
- **Affectation** : id, dateDebut, dateFin, statut, materiel, employe

## 🚀 Fonctionnalités

- 🖥️ **Matériel :** ajouter, modifier, supprimer, filtrer par type/état  
- 👨‍💼 **Employés :** ajouter, modifier, supprimer  
- 📦 **Affectations :** créer, modifier, clôturer, filtrer par service/statut  
- 📊 **Statistiques** :  
  - 🔹 Taux d’utilisation du matériel (barre de progression)  
  - 🔹 Nombre de matériels par état (tableau ou graphique)  

---

## 🛠️ Technologie utilisée

- ☕ Java 17  
- 🏗️ Spring Boot 3 + Spring Data JPA  
- 🖋️ Thymeleaf  
- 🎨 Css  
- 🗄️ MySQL pour la base de données
